# Proyecto_Final-

## Breve descripcion:
nuestro proyecto se basa en la copia de un juego de logica matematica que se trata de poner los numeros en un orden descendente (de mayor a menor) nos guiamos de un codigo base para realizar esto usando todo lo visto en clase, como por ejemplo "while" para regresar a nuestro menu, "if" para saber si el punto es valido o no y arreglos para saber los puntajes de las personas que jueguen.
